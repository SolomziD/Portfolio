# Solomzi Dhladhla - Software Developer Portfolio

A responsive, single-page portfolio website showcasing my skills, projects, and contact information.

![Logo](assets/logo.svg)

## Features

- **Responsive Design**: Adapts seamlessly to all screen sizes (Desktop, Tablet, Mobile).
- **Dark Theme**: Custom deep blue (`#0D2440`) aesthetic with high-contrast elements.
- **Interactive UI**:
  - Sticky navigation bar with blur effect.
  - Typewriter animation for the hero section.
  - Hover effects on cards, links, and buttons.
  - Smooth scrolling between sections.
- **Sections**:
  - **Hero**: Welcomes visitors with a dynamic introduction.
  - **About**: Brief professional summary.
  - **Skills**: Visual representation of proficiency in languages, frameworks, tools, and IDEs.
  - **Projects**: Showcase of past work with descriptions and links.
  - **Contact**: Functional contact form layout.

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Custom styling, Flexbox, Grid, CSS Variables, and Animations.
- **SVG**: Scalable vector graphics for the logo.

## Setup & Usage

No build process is required. Since this is a static site, you can view it directly in your browser.

1.  Clone the repository or download the files.
2.  Open `index.html` in your preferred web browser.

## Customization

The site uses CSS variables for easy theming. You can modify `css/style.css` to change the core colors and fonts:

```css
:root {
  --primary-color: #64ffda; /* Cyan accent */
  --bg-color: #0d2440; /* Background */
  /* ... */
}
```
